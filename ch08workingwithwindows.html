JavaScript includes two parent objects to control the appearance and look of a browser window: window and document. Window object is the top-level parent for all objects which includes document, location, and history. Document object allows access to the various HTML elements.

Link addresses are displayed by the browser by default and cannot be overwritten with other text.

document.bgColor change background color. document.fgColor change text color. document.linkColor change link color. document.alinkColor change active link color. document.vlinkColor change visited link color. Ch 6 Handling Events

document.title changes the title on the title bar.
